n01 Web版について
2017/05/14
--
■起動
「n01.html」をダブルクリック等でブラウザにて開くと利用できます。

■動作環境
Windowsのローカル環境で動作させる場合はGoogle ChromeかFirefoxでのみ動作します。
ローカル環境でIEで起動するとエラーとなります。

サーバに配置して動作させる場合は以下の環境で動作します。
iPhone, iPad, Android, Windows Phone, Mac, Windows(Chrome, Firefox, IE10以降)

その他、ブラウザの設定でJavascriptを有効にしておく必要があります。

■設定
タブレットで利用する場合は画面下の「・・・」を押すと出るメニューで「数字キーを
表示」を押すと数字のみのソフトキーボードが出ます。
数字キーの大きさは画面下の「・・・」を押して出るメニューの「オプション」で大き
さを変更できます。
残りスコアを大きく表示させる場合も「オプション」から設定できます。

■ゲーム設定
画面下の「New」を押すと新規ゲームの設定ができます。
以下の設定ができます。
・プレイヤー名
・開始スコア
・ラウンドを制限する場合はラウンド数
・COM対戦設定(Level 1〜12)
・スコア練習モード
  →残りスコアを自動計算せずに入力する方式になります。
    スコア練習モードは画面上部が緑色になります。
    スコアを入力した後に残り点数の入力を求められます。
    誤った残り点数を入力すると背景が赤くなります。
    過去のスコアを修正する場合は残り点数の入力は求められません。
    ESCキーで残り点数入力を補助します
・ハンディキャップ
  →プレイヤー毎に開始スコアを変更できます。
・スケジュールモード
  →セット毎にスケジューリングすることができます。
    ・セット毎に名前や開始スコア、レッグ数を設定することができます。

■スタッツ表示
画面下の「Stats」を押すと現在のゲームのスタッツが表示されます。
スタッツ画面の右上にある「履歴」を押すと過去のレッグの履歴が見れます。
